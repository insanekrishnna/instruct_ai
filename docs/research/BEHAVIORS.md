# Instruct Behaviors

- Header is fixed, centered, white, rounded, with a very subtle border. Desktop shows nav/actions; mobile shows hamburger only.
- Hero fades in over a lightly textured background image with radial/linear white wash overlays.
- CTA pill and nav buttons have soft gray hover states.
- Prompt composer is disabled/static visually, with microphone and return icons on the right.
- Category pills are static filter chips in this clone, matching the homepage default visible state.
- Feature cards use soft pastel image-like gradients, subtle borders, and hover lift.
- Cookie panel is fixed bottom-left on desktop and bottom-centered on mobile with reject/accept actions.
